Elixir Quickstart

  • Use Docker...
  • Start the REPL and get going

Run the Docker Image - REPL-Style


$ docker run -ti koenighotze/elixir

Use the REPL


Erlang/OTP 18 [erts-7.0.2] [source] [64-bit] [async-threads:10] [kernel-poll:false]

Interactive Elixir (1.0.5) - press Ctrl+C to exit (type h() ENTER for help)
iex(1)> "hello" <> " world"
"hello world"
iex(2)>  2 + 2 + 2 + 2
8
iex(3)>
Just remember, CTRL+C twice to exit.

Getting help

For example, the documentation for IO.puts can be found using:


iex(3)> h IO.puts

Keep in mind, that h is short for h/0, meaning h with no arguments.

Run the Docker Image - Development Style


$ docker run -ti  -v <YOUR_SCRIPT_DIR>:/home/demo/src \
		koenighotze/elixir \
		/bin/bash
/ #
Demo code is located in /home/demo/code

Run scripts

From the commandline


/ # cat hello.exs
IO.puts "Hello World"

/ # elixir hello_world.exs
Hello World

From the REPL


iex(1)> import_file "hello.exs"

Immutability

“GOTO was evil because we asked, how did I get to this point of execution?' Mutability leaves us with, 'how did I get to this state?'”

Everything is immutable - The End

  • Assignment is an assertion

Types

Collections - Tuples


iex(14)> {status, file} = File.open("/etc/hosts")
{:ok, #PID<0.75.0>}

iex(17)> {:ok, file} = File.open("/etc/notthere")
** (MatchError) no match of right hand side value: {:error, :enoent}

Collections - Lists


iex(30)> [head | tail] = a
[1, 2, 3]
iex(32)> ^head = 1
iex(34)> ^tail = [2, 3]

iex(37)> a ++ [4, 5, 6]
[1, 2, 3, 4, 5, 6]

iex(38)> a -- [1, 2]
[3]

iex(39)> 1 in a
true

Collections - Keyword list


iex(3)> a = [zip: 40589, city: "Düsseldorf", team: "F95"]
[zip: 40589, city: "Düsseldorf", team: "F95"]

iex(5)> [h | t] = a
[zip: 40589, city: "Düsseldorf", team: "F95"]

iex(6)> h
{:zip, 40589}

Collections - Maps


iex(11)> world = %{ :germany => [1954, 1974, 1990, 2014], :netherlands => [] }
%{germany: [1954, 1974, 1990, 2014], netherlands: []}

iex(12)> world[:germany]
[1954, 1974, 1990, 2014]

iex(13)> world.germany
[1954, 1974, 1990, 2014]

Pattern matching

Simple examples


iex(1)> a = [1, 2, 3]
[1, 2, 3]

iex(5)> [a, b] = [1, 2]
[1, 2]
iex(6)> [a, b] = [1, 2, 3]
** (MatchError) no match of right hand side value: [1, 2, 3]

iex(8)> [a] = [[1, 2, 3]]
[[1, 2, 3]]
iex(9)> a
[1, 2, 3]

iex(10)> [[a]] = [[1, 2, 3]]
** (MatchError) no match of right hand side value: [[1, 2, 3]]

iex(10)> [a, b, _] = [1, 2, 3]
[1, 2, 3]

Bindings
